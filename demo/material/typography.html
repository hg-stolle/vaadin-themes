Modular scale

The font sizes follow a modular scale, which is controlled by the following custom properties:

--material-font-scale-up
The scaling factor used for font sizes above M.
--material-font-scale-down
The scaling factor used for font sizes below M.

Emphasis

The theme defines different font weights you can use. You can apply the weights either with CSS custom properties or with HTML attributes.

You can combine the emphasis styles with the sizes.
